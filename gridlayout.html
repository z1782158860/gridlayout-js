概述
API文档

GridLayout 是一个灵活的网格布局系统，支持拖拽操作、响应式布局和各种小组件的集成。主要特性包括：

  • 支持自定义尺寸（任意cols/rows组合）
  • 底部支持拖拽移动和调整位置
  • 触摸设备支持
  • 响应式布局调整
  • 小组件集成和隔离
  • 布局导入/导出功能
  • 独立数据支持
  • 支持从URL动态加载小组件资源

尺寸配置

// 支持自定义尺寸
sizes: [
  { cols: 1, rows: 1 }, // 100×100px
  { cols: 2, rows: 1 }, // 220×100px
  { cols: 2, rows: 2 }, // 220×220px
  { cols: 4, rows: 2 } // 460×220px
]

小组件结构

const grid = new GridLayout('#grid-container');
grid.init();

//设置全局暴露数据
grid.setExposedGlobals({
  func1, //支持函数
  var1, //支持变量
  notify: window.alert, //支持重命名
  ...
});

// 示例1：直接添加小组件内容
grid.addItem(
  {cols: 2, rows: 2},
  {
    text: '示例组件',
    html: '<div>Hello World</div>',
    css: 'div { color: red; }',
    js: 'notify.info(GridLayout.globals.apiKey); //调用暴露函数
  },
  {
    apiKey: "12345", //独立暴露数据，可用于独立传递数据
  },
  {x:0,y:0}
);

// 示例2：动态加载小组件（通过URL）
grid.addItem(
  {cols: 2, rows: 2},
  {
    htmlUrl: 'widgets/dynamic.html',
    cssUrl: 'widgets/dynamic.css',
    jsUrl: 'widgets/dynamic.js',
    text: '动态加载组件'
  },
  {notify},
  {x:0,y:0}
);

作用域处理： GridLayout会自动为每个小组件的CSS和JS创建独立的作用域，避免冲突。

动态加载机制： 使用URL方式加载的资源会被异步获取并应用，系统会自动处理加载状态和错误处理。

构造函数

⚙️ new GridLayout(containerSelector)
创建GridLayout实例
containerSelector string 网格容器的CSS选择器
💡 示例：
const grid = new GridLayout('#grid-container');
grid.init();

核心方法

➕ addItem(size, widgetData = null, widgetGlobals = null, position = null)
添加一个新项目到网格布局
size object 项目尺寸 {cols: number, rows: number}
widgetData object | null 小组件数据 {text: string, html: string, css: string, js: string} 或 {htmlUrl: string, cssUrl: string, jsUrl: string, text: string}
widgetGlobals object | null 小组件独立暴露数据 {a,b,c...} //任意可序列化数据类型，可省略，一般用于给小组件传递数据
position object | null 位置信息 {x:0,y:0} //省略时自动寻找合适位置，一般在恢复布局时使用
💡 示例：
// 示例1：直接添加小组件内容
grid.addItem(
  {cols: 2, rows: 2},
  {
    html: '<div>Hello World</div>',
    css: 'div { color: red; }',
    js: 'console.log("Component loaded");'
    text: '示例组件'
  }
);

// 示例2：动态加载小组件（通过URL）
grid.addItem(
  {cols: 2, rows: 2},
  {
    htmlUrl: 'widgets/dynamic.html',
    cssUrl: 'widgets/dynamic.css',
    jsUrl: 'widgets/dynamic.js',
    text: '动态加载组件'
  }
);
➖ removeItem(itemId)
从网格中移除项目
itemId number 要移除的项目ID
💡 示例：
// 删除小组件
const grid = new GridLayout('#grid-container');
grid.init();
const itemId = grid.addItem(...);
grid.removeItem(itemId);
📋 getLayoutJson()
获取当前布局的JSON表示
const layoutJson = grid.getLayoutJson();
console.log(layoutJson);
📥 importLayout(json)
从JSON导入布局
json string 布局的JSON字符串
grid.importLayout(json);
🔄 reset()
重置布局到初始状态
grid.reset();
🌐 setExposedGlobals(globals)
设置暴露给小组件的全局对象
globals object 要暴露的全局对象
💡 示例：
// 暴露notify对象给小组件
grid.setExposedGlobals({
  notify: window.notify,
  ...
});
